# Intent

## Prompt

Build a Scientific Calculator API — a stateless HTTP API that performs scientific math operations (arithmetic, trigonometry, logarithms, powers, statistics, unit conversions) using Python 3.13, FastAPI, and uv.

## Summary

Deliver a greenfield stateless HTTP API that exposes scientific math operations — arithmetic, powers/roots, trigonometry, logarithms, statistics, constants, and unit conversions — over a JSON REST interface. The service is built with Python 3.13, FastAPI, Pydantic v2, and managed exclusively with uv. It targets correctness (matching Python `math` stdlib to ≤1 ULP), structured error reporting, and ≥90% test coverage.
